  •  I've done.

    • Dungeon Escape

    I designed and developed a 2D tile-based dungeon exploration game with randomly generated rooms and hallways. Features include keyboard interaction, save and load feature, and an interactive heads up display.

    Bear Maps

    I developed the backend of a street mapping web app for the Berkeley area inspired by Google Maps. Components include rastering, converting query requests to bit-mapped pixel-by-pixel images, and location lookup with prefix matching search.

    NBody Simulation

    This project simulates the motion of N objects on a plane. It draws an animation of bodies floating and tugging on each other with the power of gravity as demonstrated by Newton’s Law of Universal Gravitation.

    Scheme Interpreter

    Many quirks of languages are byproducts of implementation decisions in interpreters and compilers. Here, I developed an interpreter for a subset of the Scheme language.

    Ants VS. Bees

    I created a tower defense game inspired by PopCap Games’ Plants Vs. Zombies. The ant colony is populated by ants that protects the queen ant from evil bees by throwing leaves at them.
